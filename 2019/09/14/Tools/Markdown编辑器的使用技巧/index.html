Posted on 2019-09-14 | In 工具 , markdown
  • 作为新一代编辑器，Markdown更容易读、写、改，这里稍微介绍一下几个使用技巧，其中有些细节可能不会详细，具体请君们自己操作一遍，其中大有名堂，无法全部讲完。

1.标题#的使用

6个‘ # ‘对应着6级标题，每增加一个‘ # ‘字体变小一号，当有7个#一起出现时，变为7个# + 输入的字符串。

2.引用 > 的使用

在文本或照片链接前加上>,即可出现引用的格式，这里需要注意的是，当两个 > 中间没有空行时，他们合为一个引用(或者是上一个文档使用了<,下一个文档与他没有空行时，下一行文档也会在引用中）。（当然，照片和文本可以合在一个引用里！！）

3.文本链接插入 [ ]( ) 的使用

在[ ]内输入链接标志（关键名啥的），( )内输入链接网址。这样，当你在点击蓝色字部分的时候，会产生一个链接网址的的新标签页。

4.图片链接 ![]() 的使用

  • 若需将一个本地的图片插入到文本中，直接将图片拖拽到对应的位置即可，编译器会自动生成图片的链接，此时[ ]内会显示图片的名字，()内则为图片的链接；
  • 若需要将网上的图片插入，则[ ]内输入名字(不输入也行），输入后就显示在图片的下方，( )内为图片的网上链接。

    这里注意一点：网上图片的URL地址获取

5. * 的多功能使用

  • 一个 * 放在文本的最前面且与文本留有空格时，分层作用；
  • 两个 * 放在文本的一头一尾时，表将这段文本变成斜体(*与文本间不能有空格)；
  • 两个 ** （没错，就是）放在文本的一头一尾时，表将这段文本加粗(与文本间也不能有空格)；
  • 三个(或3个以上） * 单独占一行时，会形成分界线（中间可以有空格）。

6.程序员必备， `` 的使用

注意：这里的 ` 是键盘左上角与 ~ 在一个键盘格里的那个符号；

  • 在代码的一头一尾放上一个 ` ,形成专门的代码框。
