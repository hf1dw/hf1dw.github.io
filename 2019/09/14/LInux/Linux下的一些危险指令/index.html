linux危险指令

Posted on 2019-09-14 | In linux , 指令

0X01 rm

删除文件
rm -rf 强制删除文件，不会再提醒。

0X02 :(){ :|: & };:

定义一个名为 : 的函数，并不断的调用自身，直到系统崩溃。

0X03 某命令 >/dev/sda

将“某命令”的输出写到块设备/sev/sda中，并将设备中原有的数据替换掉。

0X04 mv 文件 /dev/null

将文件移动到null上，将null原本的文件覆盖掉

0X05 wget 某网址 -O- |某目录

O后面的 - 意味着标准输出，该指令表示将从某网址下载下来的脚本在“某目录”下直接执行

0X06 mkfs.ext3 /dev/sda

将设备块sda格式化为ext3类型

0X07 file

清空文件内容和命令输出记录

0X08 ^ foo ^bar

编辑先前运行的命令，将上一条命令中的foo替换成bar
