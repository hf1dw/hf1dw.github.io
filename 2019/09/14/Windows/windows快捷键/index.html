windows快捷键

Posted on 2019-09-14 | In windows , 快捷键
  • Ctrl+W
    关闭当前浏览网页
  • Ctrl+Shift+T
    恢复之前关闭了的浏览网页
  • Windows+L
    锁屏
  • Windows+D
    显示桌面，再按一次，恢复
  • Windows+E
    打开资源管理器，查找文件
  • Windows+M
    收起所有窗口
  • Ctrl+Z
    撤销键，编辑文本时也可使用
  • Windows+ =
    放大镜
  • Ctrl+C/V
    复制/粘贴
  • Ctrl+D
    添加书签
  • Alt+Tab
    切换窗口，通过Tab控制切换到哪个窗口
  • Ctrl+T
    浏览器增加标签
  • Ctrl+(Tab | shift+Tab)
    浏览器标签切换
  • Alt+F4
    关闭当前窗口的所有标签
  • Alt+D
    覆盖浏览器当前网址
