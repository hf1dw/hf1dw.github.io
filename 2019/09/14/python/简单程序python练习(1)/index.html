题目3：一个整数，它加上100后是一个完全平方数，再加上168又是一个完全平方数，请问该数是多少？

//分析：
a+100=m^2………..(1)
a+100+168=n^2………..(2)
(2)-(1),得(n+m)(n-m)=168
令i=n+m，j=n-m；
则i+j=2n，i-j=2m
又有i*j=168
得出i，j同为偶数
且i<168/2+1=85
