python3的requests模块的导入

今天在编写一个脚本时，发现自己python3的requests模块没有导入，上网搜了一下，配置好了，这里写下过程，分享一下经验。

1、先找到自己python安装目录下的pip(一般都是在Scripts下，其中的chardetect.exe开始是没有的，后来导入requests模块后才出现）


2、打开cmd窗口，使用命令行命令cd....转到pip路径下。


3、使用pip install requests命令成功导入requests模块（导入时可能有一个意外，说是出现红字，这是由于配置python环境的时候，上面的路径没有配置到PATH环境中，详细解答见：http://blog.csdn.net/dengfan666/article/details/77163350   具体说明见廖雪峰官网上的python学习的安装python，这里截个图供参考）



这里由于作者已经成功安装，所以出现了这个界面
